/*
Função soma(): cria um polinômio e percorre
p e q utilizando a ordem crescente para somar
expoentes iguais.
*/
polinomio soma(polinomio p, polinomio q) {
    No *atual_p = (No *)p;
    No *atual_q = (No *)q;
    polinomio resultado = cria();
    No *ultimo = NULL; 

    while (atual_p != NULL && atual_q != NULL){
        if (atual_p->expoente < atual_q->expoente){
            insere(&resultado, &ultimo, atual_p->coeficiente, atual_p->expoente);
            atual_p = atual_p->prox;
        }
        else if (atual_p->expoente > atual_q->expoente){
            insere(&resultado, &ultimo, atual_q->coeficiente, atual_q->expoente);
            atual_q = atual_q->prox;
        }
        else{
            float soma_coef = atual_p->coeficiente + atual_q->coeficiente;
            insere(&resultado, &ultimo, soma_coef, atual_p->expoente);
            atual_p = atual_p->prox;
            atual_q = atual_q->prox;
        }
    }

    while (atual_p != NULL){
        insere(&resultado, &ultimo, atual_p->coeficiente, atual_p->expoente);
        atual_p = atual_p->prox;
    }

    while (atual_q != NULL){
        insere(&resultado, &ultimo, atual_q->coeficiente, atual_q->expoente);
        atual_q = atual_q->prox;
    }

    return resultado;
}



Infixa: É a notação que usamos normalmente em expressões matemáticas. O operador fica entre os operandos. Exemplo: a + b.
Prefixa (ou Polonesa): O operador vem antes dos operandos. Exemplo: + a b.
Posfixa (ou Polonesa Inversa): O operador vem depois dos operandos. Exemplo: a b +.

infixaParaPosfixa
Aloca memória para a expressão posfixa (posfixa).
Inicializa a pilha (stack).
Itera sobre cada caractere da expressão infixa.
'(': Empilha na pilha.
')': Desempilha da pilha e adiciona à expressão posfixa até encontrar um '(' (que é descartado).
'+', '-': Desempilha operadores da pilha e adiciona à expressão posfixa até encontrar '(' ou a pilha ficar vazia. Então, empilha o operador atual.
'*', '/': Desempilha operadores da pilha e adiciona à expressão posfixa até encontrar '(', '+', '-', ou a pilha ficar vazia. Então, empilha o operador atual.
Operando: Adiciona diretamente à expressão posfixa.
Desempilha todos os operadores restantes da pilha e adiciona à expressão posfixa.
Libera a memória da pilha.
